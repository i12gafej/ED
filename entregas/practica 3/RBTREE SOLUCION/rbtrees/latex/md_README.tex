La práctica se estructurará en dos etapas\+:


\begin{DoxyItemize}
\item La primera etapa se debe implementar sólo las operaciones de árbol binario de búsqueda ({\ttfamily \+\_\+\+\_\+\+ONLY\+\_\+\+BSTREE\+\_\+\+\_\+})\+: localización, inserción y borrado de claves sin implementar los algoritmos de equilibrado. También se implementará la generación de un árbol binario de búsqueda perfectamente equilibrado conocidas todas las claves a insertar.
\item En la segunda etapa, se deben implementar las operaciones para obtener el árbol de búsqueda “casi” equilibrado.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysection{Evaluación}\label{md_README_autotoc_md3}
