bool RBTree< T >::check_min_max_branch_length
protected

Check Red-Black tree property.

Number of nodes from a node to its farthest descendant leaf is no more than twice as the number of nodes to the nearest descendant leaf. We test this from the root node only.

Returns
true if the property is meet.

staticnoexcept

Unfold an RBTree from an input stream. The input format will be: '[]' for the empty tree. '[ '' '<R|B>' '<left-subtree>' '<right-subtree>' ]'.

For example a tree with three nodes: "[ 2 B [ 1 R [] [] ] [ 3 R [] [] ] ]"

Parameters
inis the input stream.
parentis the parent node of the node to be unfolded.
Returns
The node unfolded on success.
Warning
std::runtime_error("Wrong input format"), std::runtime_error("It is not a binary search tree") or std::runtime_error("It is not an rbtree") will throw if an error happened.

int  dir 
)
protected

rotate to re-balanced a subtree.

Rotate left (dir == 0) 

          P      ->    S
         / \          / \
        N   S        P   D
           /  \     / \
          C    D   N   C

Rotate right (dir == 1) 

          P      ->    S
         / \          / \
        S   N        D   P
      /  \              / \
     D    C            C   N
Parameters
Pis the root node of the subtree.
diris the direction (0->left, 1->right) to rotate.
Returns
S, the new root node of the rotated subtree.
Precondition
P!=nullptr
dir in {0,1}
S==P->child(1-dir) != nullptr
