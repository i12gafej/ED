Detailed Description

template<class K, class V, class KeyToInt>
class HashTable< K, V, KeyToInt >

Implement the HashTable[K,V] ADT.

The template parameter keyToInt is a functional to transform values of type K into size_t. It must implement the interface: std::uint64_t operator()(K const&k)

Invariant
load_factor() <= 0.5
